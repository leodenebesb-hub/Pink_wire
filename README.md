# Pink Wire Procurement Site

Static one-page procurement report prepared for GitHub Pages.

## Files

- `index.html` - the published one-page site.
- `.nojekyll` - disables Jekyll processing on GitHub Pages so the file is served as-is.

## Recommended GitHub Pages settings

Use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After GitHub finishes deployment, the public URL will be:

`https://leodenebesb-hub.github.io/pink-wire-procurement-site/`

GitHub Pages can take several minutes to publish the first deployment.
